##### 第一天：系统设置无反应
<pre><code class="language-shell line-numbers">

sudo apt-get install software-center

##### 第二天：系统登录启动会话失败
按住 ctrl + Alt + F1 进入到tty，登录终端
<pre><code class="language-shell line-numbers">
sudo apt-get install ubuntu-session
#重启
sudo reboot
</code></pre>
